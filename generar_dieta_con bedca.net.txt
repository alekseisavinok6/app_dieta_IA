Pregunté Gemini (IA):
¿Es posible generar dieta con https://www.bedca.net/bdpub/?

Su respuesta era:
No, no es posible generar una dieta directamente a través del sitio web de BEDCA 
(Base de Datos Española de Composición de Alimentos). 
Este sitio web funciona como una base de datos completa que proporciona información sobre
la composición nutricional de diversos alimentos. 
Permite consultar los valores de composición de alimentos, lo cual puede ser útil para planificar
la dieta, pero no ofrece herramientas para generar planes de dieta automáticamente.

Puede acceder a la base de datos para consultar los valores de composición de alimentos haciendo clic en
"Consulta" en el sitio web:

* [Base de Datos Española de Composición de Alimentos de BEDCA](https://www.bedca.net/bdpub/)
